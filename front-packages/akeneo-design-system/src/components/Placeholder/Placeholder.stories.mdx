# Placeholder

## Usage

Placeholders are used to have a good visual representation of the items to display. Placeholders can be used in a grid or in a collection.
_Example: Assets manager or Connections._

Placeholders should be displayed in a grid, therefore, a `PlaceholderGrid` component is also available with a convenient `size` prop. Possible values for the `size` prop are: `normal` or `big`.
