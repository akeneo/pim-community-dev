# SubNavigation

## Usage

The Sub Navigation is used by the Navigation pattern. This components helps the users in their daily usage as an easy
way to discover and navigate the interface. Also, it's a companion for them to understand where they are in their
journey.

### Interaction

The panel is open by default and can be collapsed with the dedicated button.

When the panel is collapsed the content is hidden and a dropdown menu is available with the SubNavigation.Section(s) and
their SubNavigation.Item(s)
