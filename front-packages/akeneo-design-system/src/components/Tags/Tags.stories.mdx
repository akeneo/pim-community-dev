# Tags

## Usage

Tags can be used to display a list of keywords.

A `Tags` element can only accept a list of `Tag` children.

Each Tag must have one `tint` attribute that you can pick in that list: `'green' | 'blue' | 'dark_blue' | 'purple' | 'dark_purple' | 'yellow' | 'red'`
